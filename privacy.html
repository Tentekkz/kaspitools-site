Передавая Оператору персональные данные посредством Telegram-бота, Пользователь подтверждает своё согласие на использование указанных данных на условиях, изложенных в настоящей Политике.

Если Пользователь не согласен с условиями настоящей Политики, он обязан прекратить использование Telegram-бота. Безусловным акцептом настоящей Политики является начало использования Telegram-бота.

1. ТЕРМИНЫ

1.1. Telegram-бот — бот, расположенный в сети Интернет по адресу @kitseller_bot. Все исключительные права на Telegram-бот принадлежат Оператору в полном объёме.

1.2. Оператор — ИП Path of Peaks, собственник Telegram-бота @kitseller_bot.

1.3. Пользователь — лицо, использующее Telegram-бот.

1.4. Законодательство — действующее законодательство Республики Казахстан.

1.5. Персональные данные — данные Пользователя, которые Пользователь предоставляет самостоятельно при использовании функционала Telegram-бота.

1.6. Услуги — автоматизация работы продавцов на маркетплейсе Kaspi.kz: обработка накладных, аналитика продаж, калькулятор юнит-экономики, курсы валют, QR-коды.

2. СБОР И ОБРАБОТКА ПЕРСОНАЛЬНЫХ ДАННЫХ

2.1. Оператор собирает и хранит только те данные, которые необходимы для оказания Услуг и взаимодействия с Пользователем.

2.2. Персональные данные используются в следующих целях:

оказание Услуг Пользователю;

идентификация Пользователя;

взаимодействие с Пользователем (уведомления о подписке);

проведение статистических исследований для улучшения сервиса.

2.3. Оператор обрабатывает следующие данные:

Telegram ID и имя пользователя — для идентификации;

дата и статус подписки — для управления доступом;

статистика использования инструментов (без личных данных) — для улучшения качества.

2.4. Загружаемые Пользователем файлы (ZIP-архивы накладных, Excel-файлы из Kaspi Pay) обрабатываются исключительно в оперативной памяти сервера и удаляются сразу после обработки. Данные файлы не сохраняются на серверах Оператора.

2.5. Пользователю запрещается передавать через Telegram-бот персональные данные третьих лиц без их согласия.

3. ПОРЯДОК ОБРАБОТКИ ДАННЫХ

3.1. Оператор обязуется использовать персональные данные в соответствии с Законом «О персональных данных и их защите» Республики Казахстан.

3.2. В отношении персональных данных Пользователя сохраняется их конфиденциальность, кроме случаев, когда данные являются общедоступными.

3.4. Оператор вправе передавать персональные данные без согласия Пользователя только:

государственным органам по их мотивированному запросу в соответствии с законодательством;

в иных случаях, прямо предусмотренных действующим законодательством РК.

3.5. Данные не передаются третьим лицам в коммерческих целях и не используются для рекламных рассылок без согласия Пользователя.

4.1. Оператор принимает необходимые организационные и технические меры для защиты персональных данных от неправомерного доступа, уничтожения, изменения, блокирования и иных неправомерных действий.

4.2. Доступ к персональным данным имеет только Оператор. Данные хранятся на защищённых серверах.

5. ПРАВА ПОЛЬЗОВАТЕЛЯ

5.1. Пользователь вправе запросить удаление всех своих данных из системы, направив обращение на @peaks_kz или по email Оператора. Данные будут удалены в течение 7 рабочих дней.

6.2. Все споры разрешаются в соответствии с законодательством по месту регистрации Оператора. Обязательный досудебный порядок: претензия в письменном виде, срок ответа — 30 рабочих дней.
