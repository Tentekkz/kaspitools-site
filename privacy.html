Политика конфиденциальности

и обработки персональных данных Telegram-бота KaspiTools (@kaspitoolsbot) · Дата: 01.04.2026

🔒 Мы не продаём и не передаём ваши данные третьим лицам. Загружаемые файлы обрабатываются только в памяти сервера и удаляются сразу после обработки.

Настоящая Политика регулирует порядок обработки персональных данных Telegram-бота @kaspitoolsbot (далее — Оператор). Действующая редакция доступна в боте по команде /privacy и на сайте kaspitools.kz/privacy.html

Начало использования Telegram-бота означает безоговорочный акцепт настоящей Политики.

1. Термины

Telegram-бот — бот @kaspitoolsbot. Все исключительные права принадлежат Оператору.

Оператор — ИП Path of Peaks, собственник Telegram-бота @kaspitoolsbot.

Пользователь — лицо, использующее Telegram-бот.

Персональные данные — данные, которые Пользователь предоставляет при использовании бота.

Услуги — автоматизация работы продавцов на Kaspi.kz: накладные, аналитика, калькулятор, курсы валют, QR-коды.

2. Сбор и обработка данных

2.1. Оператор собирает только данные, необходимые для оказания Услуг.

2.2. Цели обработки: оказание Услуг, идентификация Пользователя, уведомления о подписке, улучшение сервиса.

2.3. Оператор обрабатывает: Telegram ID и имя (идентификация), дата и статус подписки (управление доступом), статистика использования без личных данных (улучшение качества).

2.4. Загружаемые файлы (ZIP, XLSX) обрабатываются исключительно в оперативной памяти и удаляются сразу после обработки. На серверах не сохраняются.

2.5. Запрещается передавать через бот персональные данные третьих лиц без их согласия.

3. Порядок обработки данных

3.1. Данные обрабатываются в соответствии с Законом «О персональных данных и их защите» РК.

3.2. Данные Пользователя конфиденциальны, кроме общедоступных случаев.

3.4. Оператор передаёт данные без согласия только государственным органам по мотивированному запросу либо в иных случаях, предусмотренных законодательством РК.

3.5. Данные не передаются третьим лицам в коммерческих целях.

4. Защита данных

4.1. Оператор принимает организационные и технические меры для защиты данных от неправомерного доступа и иных неправомерных действий.

4.2. Доступ к персональным данным имеет только Оператор.

5. Права пользователя

5.1. Пользователь вправе запросить удаление всех своих данных — обратитесь на @peaks_kz или по email. Данные будут удалены в течение 7 рабочих дней.

5.2. Пользователь вправе запросить информацию о составе хранимых данных.

6. Иные положения

6.1. Применяется право Республики Казахстан.

6.2. Споры разрешаются по месту регистрации Оператора. Досудебный порядок: письменная претензия, срок ответа — 30 рабочих дней.

6.3. Оператор вправе изменить Политику в одностороннем порядке. Изменения вступают в силу с момента публикации в боте. Продолжение использования = согласие.

6.4. Вопросы: или @peaks_kz
